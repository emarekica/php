Documentation: https://www.php.net/

---

## First steps

1. Make `index.php`.


```php
<?php

echo 'Hello, world';
```

2. See the file in the terminal:

```php
php index.php
// prints: 'Hello, world'
```

3. Run local server which comes with PHP.

```php
php -S // runs the server

php -S localhost:8000
```

4. Stop the server: `control C`
